# InternSpark-Internship
Repository containing tasks, projects, reports, and learning outcomes completed during my Task-Based VLSI Internship with InternSpark. This repository showcases practical work in digital design, Verilog, RTL design, FPGA concepts, and related VLSI topics.
# InternSpark Internship — Digital VLSI Design Flow

## Objective
The core objective of this internship is to gain comprehensive, hands-on experience across the entire digital VLSI design flow. This includes everything from RTL design and functional simulation to synthesis, verification, and basic physical design checks, with a strong emphasis on low-power design, timing closure, and design for testability.
---
## Technologies and EDA Tools
This repository utilizes industry-standard hardware description languages and Electronic Design Automation software:
* Languages: Verilog / SystemVerilog
* Simulation and Verification: ModelSim / VCS / SPICE / NGSPICE
* Synthesis and Timing: Cadence / Synopsys / Primetime (STA)
